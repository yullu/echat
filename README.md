## About echat

This echat is a laravel 12 application running on livewire and reverb.

## Features

- Login which is powered by breeze starter kit
- chat feature which incorporates livewire and reverb which is a first-party WebSocket server for Laravel applications, bringing real-time communication between client and server directly to your fingertips. Open source and an Artisan command away - with love from the Laravel team.
